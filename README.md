# Bookme

A ticket booking micro-service application

![architecture](./images/architecture.png)

## Tech Stack

- **Server**: ExpressJS
- **Client**: NextJS
- **Database**: MongoDB, Redis
- **Event Streaming**: NATS (Event Bus)
- **Dev-Ops**: Docker, Kubernetes, Skaffold
- **Testing**: Jest
